Photo_Patchwork:
  Contributors:
    - Monika Smaza
    - Kamil Kucharski
    - Piotr Majkut
  Project Purpose:
    - program allows user to make a collage picture using segments of others pictures
  Used technologies:
    - C++ programming language
    - wxWidgets framework


Interakcja użytownika z programem:
----------------------------------
1. Dwukrotne klinknięcie lewege przycisku myszy na miniaturze - załadowanie danej miniaturki na główny panel.
2. Dwukrotne klinknięcie lewege przycisku myszy na głównym panelu - wybór kolejnych punktów ograniczajšcych wycinany fragment.
3. Dwukrotne klinknięcie prawego przycisku myszy na głównym panelu - zakończenie wybierania punktów oraz przeklejenie wybranego fragmentu na tło.
